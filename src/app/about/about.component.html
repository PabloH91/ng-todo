About this app

Why

I'm trying to understand the latest versions of Angular and Bootstrap. It wasn't going too well, so I decided to make a smol for me to practice these things with. Wish me luck.

What

A nicer breakdown:

  • Angular's CLI. I've tried to create the entire app using the CLI to let it implement best practices to learn from
  • Modules, found for the base app and the tasks child module (which is lazy loaded)
  • Routing, found as forRoot in app module and as forChild in the tasks module
  • Route Guards, found when trying to access a route that does not exist
  • Components, found in the different pages developed
